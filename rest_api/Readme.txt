1. Need to install node
2. After that open cmd and go to restapi folder
3. Then run npm install
4. then run npm run dev
5. Server will be up in 3005 port
6. open http://localhost:3005/v1/route/get?url=https://www.wiprolimited.com&fileName=test19.txt
7. first parameter is URL and second is file name to write the output.
8. output will be stored in root folder.
